# BOB SDK

The BOB SDK helps you interact with BOB and Bitcoin, including Ordinals, BRC20, Runes, and more.

## Learn more

- [Website](https://www.gobob.xyz/)
- [Docs](https://docs.gobob.xyz/)

## Installation

```shell
pnpm i @bob-sdk/sdk
```

## Test

```shell
pnpm test
```
